# Smart Bin Mobile Application

This repository contains the mobile application and backend services
for a smart garbage bin that sorts waste into:

- Organic
- Glass
- Plastic
- Cans
- Mixed / General

## Tech Stack
- Flutter (Mobile + Web)
- Firebase Firestore (Database)
- Firebase Cloud Functions (API)
- Firebase Cloud Messaging (Notifications)
